TRUSDX-AI DRIVER - QUICK INSTALL GUIDE
======================================

PROVEN WORKING - CONTACT MADE!

1. CONNECT TRUSDX
   - Connect truSDX to computer via USB cable

2. EXTRACT FILES
   - Extract this folder to your desired location
   - Open terminal in this folder

3. GUI REQUIREMENTS
   - Install GUI packages: sudo apt install python3-tk python3-matplotlib
   - Required for VU meter and waterfall displays
   - Use --nogui flag to disable visuals if not needed

4. RUN SETUP
   - Make setup script executable: chmod +x setup.sh
   - Run setup: ./setup.sh
   - Enter password when prompted for sudo

5. START DRIVER
   - Run: ./trusdx-txrx-AI.py
   - Leave terminal open while using

6. CONFIGURE WSJT-X
   - Radio: Kenwood TS-480
   - Serial Port: /tmp/trusdx_cat
   - Baud: 115200
   - Audio In: Monitor of TRUSDX
   - Audio Out: TRUSDX
   - PTT: CAT
   - Poll: 80ms

7. TEST
   - Start WSJT-X
   - Check VU meter shows activity
   - Try changing frequency/band
   - Make a contact!

TROUBLESHOTING:
- If no audio: run 'pavucontrol' and check routing
- If no CAT: check USB connection and restart driver
- For help: run with --verbose flag

Enjoy and 73!
